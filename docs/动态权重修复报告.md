## 📋 问题分析

### 发现的问题

在之前的修复中，我将综合评分公式简化为固定的均等权重：
```
【综合评分】：X/100
计算公式：(技术20% + 资金20% + 基本面20% + 新闻20% + 社区情绪20%)
```

但是在**多维分析师Agent定义**（`src/agents/smart_agents.py`第2780-2856行）中，明确要求：

### 正确的权重调整方法

#### 环境1：牛市 + HOT市场
```
技术面权重：30%（技术指标更可靠）
资金面权重：30%（资金活跃，量价配合重要）
基本面权重：15%（基本面次要）
新闻面权重：10%（新闻炒作多）
社区情绪权重：15%（散户情绪反向指标）

综合评分 = 技术面×0.30 + 资金面×0.30 + 基本面×0.15 + 新闻面×0.10 + 社区情绪×0.15
```

#### 环境2：熊市 + COLD市场
```
技术面权重：15%（技术指标可能失效）
资金面权重：30%（资金流向是关键）
基本面权重：35%（基本面是安全边际）
新闻面权重：10%（新闻影响有限）
社区情绪权重：10%（散户悲观是买点）

综合评分 = 技术面×0.15 + 资金面×0.30 + 基本面×0.35 + 新闻面×0.10 + 社区情绪×0.10
```

#### 环境3：震荡市 + WARM/NEUTRAL市场
```
技术面权重：25%（关注支撑位和压力位）
资金面权重：25%（资金流向决定方向）
基本面权重：25%（基本面是选股基础）
新闻面权重：15%（新闻可能引发波动）
社区情绪权重：10%（情绪分歧是机会）

综合评分 = 技术面×0.25 + 资金面×0.25 + 基本面×0.25 + 新闻面×0.15 + 社区情绪×0.10
```

### Agent定义中的要求

从`src/agents/smart_agents.py`第2979-2981行：
```python
**第5步**：计算综合评分
- 默认权重：综合评分 = (技术+资金+基本面+新闻+社区情绪) / 5
- 动态权重：综合评分 = 技术×权重1 + 资金×权重2 + 基本面×权重3 + 新闻×权重4 + 社区情绪×权重5
```

**关键点**：需要输出**两个评分**：
1. **默认权重评分**（均等权重）
2. **动态权重评分**（根据市场环境调整）
