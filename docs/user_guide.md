# 用户使用指南 (User Guide)

欢迎使用 CrewAI Stock 智能交易系统。本指南将帮助你快速上手，从登录系统到配置实盘交易。

## 🖥️ Web 界面概览

系统提供了一个现代化的 Web 管理界面，访问地址通常为 `http://localhost:5000`。

### 1. 仪表盘 (Dashboard)
- **资产概览**：显示当前总资产、持仓市值、可用资金。
- **今日推荐**：展示 AI 今日最新推荐的股票及其评分。
- **持仓列表**：显示当前持仓的实时盈亏、AI 建议（持有/卖出）。
- **数据统计**：历史推荐胜率、策略分布图表。

### 2. 登录系统
系统采用简单的用户名机制进行隔离：
- 首次访问会跳转到登录页。
- 输入任意用户名（建议使用英文+数字，如 `user001`）。
- **注意**：用户名即是你的身份标识，不同用户名的数据是完全隔离的。请记住你的用户名以便下次访问同一份数据。

## ⚙️ 策略配置与运行

### 手动触发推荐
虽然系统有定时任务，但你也可以随时手动触发推荐：
1. 点击界面右上角的 **"AI 推荐"** 按钮。
2. 选择推荐模式：
   - **快速扫描**：速度快，分析深度一般。
   - **深度研报**：速度慢（约3-5分钟），生成详细的分析报告。
3. 等待进度条完成，结果会自动弹窗显示。

### 策略偏好设置
AI 会根据市场情况自动选择策略，但你也可以通过 "记忆" 功能干预：
- 在对话框或设置中输入："请记住：我偏好稳健型策略，只关注沪深300成分股"。
- 系统会将此偏好存入 User Memory，后续 Agent 在决策时会优先考虑。

## 📈 实盘交易设置 (可选)

本系统支持对接 Windows 版同花顺客户端进行实盘交易。

### 前置条件
1. 一台 Windows 电脑。
2. 已安装并登录 **同花顺独立下单程序** (xiadan.exe)。
3. 保持同花顺客户端在前台或最小化运行（不能关闭）。

### 启用步骤
1. 在 `.env` 文件中配置：
   ```ini
   TRADING_ENABLED_USERS=你的用户名,user2
   ```
   只有在名单中的用户才会触发实盘接口。

2. 自动同步持仓：
   - 系统启动时会自动从同花顺同步持仓到数据库。
   - 你也可以在 Web 界面点击 **"同步持仓"** 按钮手动同步。

### 自动交易逻辑
- **买入**：当 AI 推荐股票且评分 > 85 分，且资金充足时，会自动挂单买入。
- **卖出**：当持仓监控发现 "强烈建议卖出" 信号（如触发止损或盘口恶化）时，会自动挂单卖出。
- **注意**：自动交易涉及真金白银，建议初期仅开启 "模拟盘" 模式或在代码中注释掉最后的 `buy/sell` 调用进行观察。

## 📱 微信通知配置

为了不错过关键交易信号，建议配置 PushPlus 微信推送。

1. 访问 [PushPlus官网](http://www.pushplus.plus/) 注册并获取 Token。
2. 修改 `.env` 文件：
   ```ini
   PUSHPLUS_TOKEN=你的Token
   PUSHPLUS_TOPIC=你的群组编码(可选)
   ```
3. 系统将在以下情况发送通知：
   - 每日 AI 推荐报告生成时。
   - 持仓股票出现紧急卖出信号时。
   - 实盘交易成交时。

## ❓ 常见问题

**Q: AI 推荐的股票一定会涨吗？**
A: **绝对不是。** AI 只是基于历史数据和当前指标进行概率分析，市场受多种不可控因素影响。请将 AI 建议作为辅助参考，而非唯一依据。

**Q: 为什么有时候获取不到数据？**
A: 可能是 API 速率限制或网络问题。系统会自动重试并切换备用源。如果持续失败，请检查网络连接或 API Key 余额。

**Q: 如何清空我的数据？**
A: 目前 Web 界面暂未提供一键重置。可以在数据库中删除 `session_id` 为你用户名的所有记录。
