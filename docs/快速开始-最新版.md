# 同花顺交易快速开始（最新版）

## ⚠️ 重要说明

**easytrader 新版本 API 变化**：
- ✅ 使用 `connect()` 方法连接客户端
- ❌ 不再使用 `prepare()` 方法自动登录
- ✅ 更安全：无需在配置文件中存储密码
- ⚠️ **必须先手动启动并登录同花顺客户端**

---

## 🚀 5分钟快速接入

### 第一步：下载同花顺客户端

**下载地址**：
- 通用版：http://www.10jqka.com.cn/
- 券商版：从您的券商官网下载（推荐）

**安装步骤**：
1. 下载安装包
2. 安装到任意目录（记住路径）
3. 记录 `xiadan.exe` 的完整路径
   - 例如：`C:\同花顺软件\xiadan.exe`

---

### 第二步：安装 Python 依赖

```bash
# 进入项目目录
cd d:\dev\github\crewAi_stock

# 安装 easytrader
pip install easytrader

# 安装 tesseract-ocr（可选，用于验证码识别）
# Windows: 下载 https://github.com/UB-Mannheim/tesseract/wiki
```

---

### 第三步：配置文件

```bash
# 1. 复制配置模板
copy ths_config.json.example ths_config.json

# 2. 编辑配置文件
notepad ths_config.json
```

**配置内容**（只需要 exe_path）：
```json
{
  "exe_path": "C:\\同花顺软件\\xiadan.exe"
}
```

**⚠️ 注意**：
- 路径使用双反斜杠 `\\`
- 不再需要 `user` 和 `password` 字段

---

### 第四步：启动并登录客户端

**这是最关键的一步！**

1. **手动启动客户端**
   ```
   双击运行：C:\同花顺软件\xiadan.exe
   ```

2. **手动登录**
   - 输入账号密码
   - 完成登录
   - **保持客户端运行**

3. **确认登录成功**
   - 可以看到持仓、资金等信息
   - 客户端保持在后台运行

## ❌ 常见错误

### 错误1：找不到进程

```
错误信息：找不到指定的进程
```

**原因**：同花顺客户端未运行

**解决**：
1. 手动启动 `xiadan.exe`
2. 确认客户端正在运行
3. 重新运行测试

---

### 错误2：路径错误

```
错误信息：配置文件中缺少 exe_path 参数
```

**解决**：
1. 检查 `ths_config.json` 是否存在
2. 检查 `exe_path` 字段是否正确
3. 路径使用双反斜杠 `\\`

---

### 错误3：连接超时

```
错误信息：连接超时
```

**原因**：客户端未登录或响应慢

**解决**：
1. 确认客户端已登录
2. 等待客户端完全启动
3. 重新运行测试

## 🎮 使用示例

### 示例1：独立运行

```bash
# 1. 启动并登录同花顺客户端
# 2. 运行交易管理器
python trade_manager.py

# 3. 按菜单操作
# 1-查询资金 2-查询持仓 3-买入 4-卖出
```

### 示例2：Python 代码

```python
from src.brokers.ths_trader import ThsTrader

# 1. 确保同花顺客户端已启动并登录

# 2. 创建交易接口
trader = ThsTrader('ths_config.json')

# 3. 查询资金
balance = trader.get_balance()
print(f"可用金额: {balance['data']['可用金额']}")

# 4. 查询持仓
positions = trader.get_position()
print(f"持仓数量: {len(positions['data'])}")

# 5. 买入股票（小额测试）
result = trader.buy('600000', 10.50, 100)
print(result)

# 6. 断开连接
trader.disconnect()
```
