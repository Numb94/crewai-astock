# 🎮 同花顺交易接入完整使用流程

## 📋 流程总览

```
下载客户端 → 安装依赖 → 配置账号 → 测试连接 → 选择集成方式 → 启用实盘
```

---

## 第一步：下载同花顺客户端 ✅

### 需要下载吗？
**是的，必须下载！** easytrader 通过 UI 自动化控制客户端。

### 下载地址

**方式1：通用版**
- 官网：http://www.10jqka.com.cn/
- 下载：http://download.10jqka.com.cn/index/download/id/113/

**方式2：券商定制版（推荐）**
- 从您的券商官网下载
- 搜索 "券商名称 + 同花顺"

### 安装步骤
1. 下载安装包
2. 安装到任意目录（记住路径）
3. 手动登录一次，确保可以正常交易
4. 记录 `xiadan.exe` 的完整路径

**示例路径**：
- `C:\同花顺软件\xiadan.exe`
- `D:\Program Files\同花顺\xiadan.exe`

## 第三步：配置同花顺账号 ✅

```bash
# 1. 复制配置模板
copy ths_config.json.example ths_config.json

# 2. 编辑配置文件
notepad ths_config.json
```

**填入您的信息**：
```json
{
  "user": "你的同花顺账号",
  "password": "你的交易密码",
  "exe_path": "C:\\同花顺软件\\xiadan.exe",
  "comm_password": "",
  "account_type": "0"
}
```

**⚠️ 注意**：
- `password` 是**交易密码**，不是登录密码
- `exe_path` 使用双反斜杠 `\\`
- 配置文件不要提交到 Git

## 第五步：选择集成方式 🎯

### 方式A：独立运行（推荐新手）

**适合场景**：
- 快速测试
- 手动交易
- 不需要集成到系统

**使用方法**：
```bash

python trade_manager.py
```

**功能菜单**：
```

2. 查询持仓信息
3. 查询委托记录
4. 买入股票
5. 卖出股票
6. 撤单
0. 退出
```

**优点**：
- ✅ 简单易用
- ✅ 无需修改代码
- ✅ 适合测试

**缺点**：
- ❌ 不能与 Web 界面集成
- ❌ 不能自动交易

### 方式B：集成到 Flask API（完整集成）

**适合场景**：
- 需要 Web 界面操作
- 需要 AI 自动交易
- 完整的系统集成

**实施步骤**：

#### 1. 修改 position_api.py

在文件顶部添加：
```python
import os
from src.brokers.ths_trader import ThsTrader
from loguru import logger

# 实盘交易配置
ENABLE_REAL_TRADING = os.getenv('ENABLE_REAL_TRADING', 'false').lower() == 'true'

# 初始化交易接口
ths_trader = None
if ENABLE_REAL_TRADING:
    try:
        ths_trader = ThsTrader()
        logger.info("✅ 实盘交易模式已启用")
    except Exception as e:
        logger.error(f"❌ 实盘交易初始化失败: {e}")
        ths_trader = None
else:
    logger.info("ℹ️ 模拟交易模式")
```

#### 2. 修改买入接口

在买入函数中添加（在创建持仓记录之前）：
```python
# 实盘交易逻辑
if ENABLE_REAL_TRADING and ths_trader:
    logger.info(f"🔴 实盘交易: 买入 {stock_code} {quantity}股 @{buy_price}")
    
    real_result = ths_trader.buy(
        stock_code=stock_code,
        price=float(buy_price),
        amount=quantity
    )
    
    if not real_result['success']:
        return jsonify({
            "success": False,
            "message": f"实盘交易失败: {real_result['message']}"
        }), 500

# 继续原有的数据库记录逻辑...
```

#### 3. 修改卖出接口

类似买入接口，添加实盘交易逻辑。

#### 4. 配置 .env

```bash
# 实盘交易开关（默认关闭）
ENABLE_REAL_TRADING=false

# 同花顺配置文件路径
THS_CONFIG_PATH=ths_config.json
```

**详细代码示例**：查看 `docs/集成到Flask示例.md`
